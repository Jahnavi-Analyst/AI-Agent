# AI-Agent
Luna is an AI-powered chatbot built for SABS College. It answers questions about admissions, courses, events, placements, facilities, students, contact information, and college details. It supports text and voice input, detects emotions with emojis, and translates different languages into English.

**Technologies Used**
•	Backend: Python (Flask), Speech Recognition, Deep Translator, Text Blob

•	Frontend: HTML, CSS, JavaScript

**Key Features**
•	Text and voice chat

•	Emotion detection with emojis

•	Language translation

•	Quick question options

•	Mobile and browser friendly

**How It Works**
•	Converts voice input to text

•	Translates text from other languages to English

•	Analyzes user's emotions

•	Sends smart responses with appropriate emojis
